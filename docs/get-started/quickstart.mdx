The only way now to launch Activepieces locally or on a cloud instance is to use our Docker image.

You can get up and running in 5 minutes, follow these instructions:

**1. Clone Activepieces repository.**

```bash
git clone https://github.com/activepieces/activepieces.git
```

**2. Go to the repository folder.**

```bash
cd activepieces
```

**3. Run Activepieces.**
```bash
docker compose -p activepieces up
```

**4. Head to https://localhost:8080 on the browser and login with the following credentials:**

|  |  |
| -------- | ---- |
| Email | admin@activepieces.com |
| Password | password |

Enjoy automating your work!
