The `docker-compose.dev.yml` file is a Docker Compose file specifically used for development environments. This file includes the necessary configurations for running a development version of your application.

Here are the steps to use `docker-compose.dev.yml`:

<Tip>
If you're using a macOS or arm based computer and using it for the first time, you should remove the node_modules folder.
</Tip>


```bash
rm -rf node_modules
```

2. Run the following command to start the development environment:

```bash
docker compose -f docker-compose.dev.yml up -d
```
<Warning>
If this is your first time running it, please note that installing node_modules may take approximately 5 minutes. Please refer to the logs for progress updates.
</Warning>

3. Go to localhost:4200 on your web browser and sign in with these details:

Email: `dev@ap.com`

Password: `12345678`
