## Prerequisites
- Node.js v18+
- npm v9+

## Instructions

1. Setup the environment
```bash
node tools/setup-dev.js
```

2. Start the environment

```bash
npm start
```

<Warning>
**Troubleshoot 1:**
The IP address 0.0.0.0:5432 is already in use. This suggests that another PostgreSQL on your device is currently active, leading to a port conflict.

Please stop the existing service or edit the environment variables in `packages/backend/.env`
</Warning>

<Warning>
**Troubleshoot 2:**
The IP address 0.0.0.0:6379 is already in use. This suggests that another Redis on your device is currently active, leading to a port conflict.

Please stop the existing service or edit the environment variables in `packages/backend/.env`
</Warning>
